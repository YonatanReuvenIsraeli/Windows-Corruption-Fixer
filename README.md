# Windows Corruption Fixer v13.2.4
Fixes corrupted Windows.

## The following options are available:
**1.** CHKDSK check.  
**2.** CHKDSK check and fix.  
**3.** View CHKDSK logs.  
**4.** DISM check health.  
**5.** DISM scan health.  
**6.** DISM restore health.  
**7.** DISM revert pending actions.  
**8.** View DISM logs.  
**9.** SFC scan now.  
**10.** SFC verify only.  
**11.** SFC scan file.  
**12.** SFC verify file.  
**13.** View SFC logs.
