# Windows Corruption Fixer v6.0.9
Fixes corrupted Windows using CHKDSK, DISM and SFC commands.
## The following options are available
**1.** Check Disk  
**2.** DISM Check Health  
**3.** DISM Scan Health  
**4.** DISM Restore Health  
**5.** View DISM Logs  
**6.** SFC Scan Now  
**7.** SFC Verify Only  
**8.** SFC Scan File  
**9.** SFC Verify File  
**10.** View SFC Logs
