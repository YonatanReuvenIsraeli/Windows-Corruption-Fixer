# Windows-Corruption-Fixer
Fixes corrupted Windows useing CHKDSK, DISM and SFC commands.
## The following options are available
**1.** SFC Scan Now  
**2.** SFc Verify Only  
**3.** SFC Scan File  
**4.** SFC Verify File  
**5.** SFC Logs  
**6.** DISM Check Health  
**7.** DISM Scan Health  
**8.** DISM Restore Health  
**9.** DISM Logs
