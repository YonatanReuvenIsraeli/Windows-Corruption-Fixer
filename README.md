# Windows Corruption Fixer v8.6.1
Fixes corrupted Windows using CHKDSK, DISM and SFC commands.
## The following options are available
**1.** Check Disk 
**2** View Check Disk Logs
**3.** DISM Check Health  
**4.** DISM Scan Health  
**5.** DISM Restore Health  
**6.** View DISM Logs  
**7.** SFC Scan Now  
**8.** SFC Verify Only  
**9.** SFC Scan File  
**10.** SFC Verify File  
**11.** View SFC Logs
